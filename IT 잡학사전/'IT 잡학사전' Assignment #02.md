- 새 프로그래밍 언어를 쉽고 빠르게 공부하는 방법 - 에피소드 3

  1. 공식 문서 살펴보기: 그 언어를 만든 사람의 철학, 누가 이 언어를 사용해야 하는지, 올바르게 사용한 예, 관련 커뮤니티 등 확인하기
  2. 문법 확인하기: 문법을 보고, 입력하고, 이해하고, 기억하려고 노력하기
  3. 다른 언어와 비슷한 특징 집중해서 보기: 내가 알고 있는 언어와 새 언어의 공통점과 차이점 살펴보기
  4. 새 언어로 코딩 시도하기: 원래 알고 있던 언어 지식을 바탕으로 함수나 변수를 선언하는 등 특정 패턴을 새 언어로 작성해보기
  5. 튜토리얼 시작하기: 수동적으로 따라하지 않고 오픈소스 프로젝트를 가져와서 해당 튜토리얼을 참고삼아 해체하고 다시 만들기
  6. 나만의 프로젝트 만들기: 이미 만들어봤던 주제의 프로젝트를 새 언어로 만들기

- C, C++, C#은 이름처럼 비슷할까? - 에피소드 5

  - C: 메모리 자원을 효율적으로 쓰면서도 최대 성능을 내는 프로그램을 만들 수 있어야 한다는 것을 최우선으로 함

  - C++: C언어를 포함하지만, C 언어는 절차 지향을 지원하고, C++ 언어는 절차 지향, 객체 지향을 모두 지원함

  - C#: 마이크로소프트가 닷넷이라는 윈도우 프로그램을 개발하기 위한 도구 또는 실행 환경을 제공하는데, 여기서 C# 언어를 사용함. 다양한 분야에서 사용 가능함. 실행 속도는 세 언어 가운데 가장 느림
